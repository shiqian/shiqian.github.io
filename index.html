从工程师到HR

缘起

作为一名理科女生，在从事攻城狮（相比“程序猿”，更爱“攻城狮”这个称号）5年后，我终于决定遵从自己的本心，向理想的职业—HR进发。

受身为语文教师的父亲的熏陶，从小我就爱阅读人文社科方面的书籍，在求学／工作过程中，我也是尽一切努力抓住让我显露写作／沟通才华的机遇。

你也许会问，当初高考填志愿你为什么不选择人力资源呢？是啊，为什么我选了电子信息工程呢？且听我慢慢道来。我高中时代的偶像，一位比我高一届的学姐，被保送清华大学电子工程系，我当时年纪小嘛，顿时觉得女生读理科好酷，恰好数学连考三次150，洋洋自得得不行，所以我在高二义无反顾选了理科，自然而然地高考后也就义无反顾地选了偶像的专业。

在工作中，我遇到过糟糕的HR，也遇到过优秀的HR。遇到前者我就会想，如果将来我从事HR了一定做得比你好；遇到后者我就会想，原来HR真的为支撑部门做出了这么大的贡献，可以让研发部门心无旁骛地工作。在参加工作后的第二年我就萌生了转岗的想法，却一直因为不自信而犹豫踌躇。天性活泼，乐于助人，善解人意，幽默诙谐，真诚豁达，这是朋友同事们为我贴的标签，从性格来看，我实在太适合HR这份职业了。但从专业来看，我需要恶补。直到数月前某天我福至心灵，立志在30岁之前转行，让人生不留遗憾，由此开启了从攻城狮到HR的漫漫征程。

structure of iptables

iptables contain multiple tables: filter table, nat table, mangle table and raw table.
filter is default table for iptables. filter table has the following built-in chains:

  • INPUT chain - incoming to firewall. for packets coming to the local server.
  • OUTPUT chain - outgoing from firewall. for packets generated locally and going out of the local server.
  • FORWARD chain -packet for another NIC on the local server. for packets routed through the local server.
